# 社区行为准则 · Code of Conduct

Ruyi 是面向真实工作的本地 AI 工具。我们希望这里是一个让不同经验、背景和观点的贡献者都能安心协作的社区。

## 我们期望的交流方式 · Expected behavior

- 讨论产品、证据和实现，不攻击个人；欢迎不同意见，也请说明理由。
- 尊重隐私、时间和安全边界；不要发布他人的个人信息、凭据或未修复漏洞。
- 使用清楚、可复现、对新参与者友好的表达；中文和英文都欢迎。
- 接受善意的代码审查与维护建议，并在发现错误时协作修正。

## 不可接受的行为 · Unacceptable behavior

- 骚扰、歧视、侮辱、威胁或持续的破坏性争论。
- 发布仇恨、色情、暴力煽动或违法内容。
- 未经许可公开私人信息、访问凭据、内部材料或安全漏洞细节。
- 用 Issue、PR、讨论或自动化工具进行垃圾信息、恶意推广或干扰维护工作。

## 适用范围与处理 · Scope and enforcement

本准则适用于本仓库的 Issue、Pull Request、讨论、提交信息及与项目相关的公开互动。维护者会根据影响和上下文采取提醒、编辑/删除内容、临时限制参与或永久禁止参与等适当措施。

如需私密报告违反本准则的行为，请通过本仓库的 GitHub Security Advisory 联系维护者，并在标题中注明「Code of Conduct」。未修复的安全问题请遵循 [`SECURITY.md`](./SECURITY.md) 的专用流程。
