# 调研与综合

适用于外部事实、选型、政策、市场或需要多来源核对的问题。先定义问题与时间边界，再检索；资料数量不是目标，可信结论才是。

## 工作流程

1. 把需求拆成 3–7 个可回答的子问题，标出地区、时间点、版本和必须满足的约束。
2. 优先官方文档、法规原文、论文、厂商规格和第一方数据；新闻或博客只作补充。
3. 每个关键结论至少找到一个直接支撑来源；高风险或有争议结论尽量交叉核对。
4. 记录来源标题、URL、发布日期/版本、它支持的具体主张和限制。
5. 主动找反例、已知缺点、样本偏差和过期信息；事实、观点与推断分开写。
6. 最终按“结论 → 证据 → 影响 → 建议”综合，不堆砌搜索摘要。

## 质量门

- 没有来源支撑的数字不写成事实。
- 对会变化的信息标注查询日期。
- 来源互相冲突时展示分歧并解释取舍，不擅自平均。
- 无法访问正文时明确标为“仅见摘要/标题”，降低置信度。

## 输出

先给简短结论，再给对照表和来源链接，最后列不确定项与下一步验证建议。
