# 如意 Ruyi 用户手册

> 面向不写代码的日常用户。读完这一本，你就能用如意工作台把「合并表格、批量改名、扫描件转文字、写周报」这类活儿交给 AI 去做，而且每一步都看得懂、都能反悔。
>
> 本手册按当前界面实况撰写；如果你的界面某处措辞与这里略有出入，**以界面实际显示为准**（软件在持续打磨）。

---

## 1. 三分钟上手

### 第一步 · 打开工作台

双击 `Start-Workbench.cmd`（或由管理员配好的桌面快捷方式）。稍等片刻，浏览器会自动打开如意工作台的界面。左上角能看到「如意 Ruyi」的字样和一枚青花色的如意图标，说明本地服务已经起来了。

如果浏览器没自动开，手动访问地址栏里管理员告诉你的本机地址即可（默认是本机的 8765 端口）。

### 第二步 · 首跑引导（第一次打开时）

第一次使用、还没有任何会话、也没设过工作文件夹时，中间会出现一张**引导卡**，自上而下三件事：

1. **把文件夹拖进来** —— 中间有个大大的「📁 把文件夹拖进来，或点击选择」区域。把你要处理的文件夹（比如装着一堆 Excel 的那个）直接拖到这里，或者点它用系统的文件夹选择器挑一个。**选中的这个文件夹就是 AI 的工作区，它只会看到这里面的东西**，别的地方碰不到。
2. **引擎状态一眼判断** —— 下面一行告诉你 AI 引擎准备好没有：
   - 绿点 +「AI 引擎已就绪：<引擎名>」= 可以开工。
   - 暖色提示卡「还没有可用的 AI 引擎」+「去设置」按钮 = 需要先配一下引擎（见第 5 章，或找管理员）。
3. **任务卡试一单** —— 引导卡下方是一排**任务卡**（合并 Excel、批量重命名、写周报……）。点其中一张，填两个空（哪个文件夹、结果存成什么名字），它会自动帮你把一段清楚的指令填好，你只管发送。这是最省心的起步方式。

> 首次使用时，安全等级默认是最稳的「每步都问」——AI 每做一个动作都先征求你同意，你可以放心试。

### 第三步 · 第一句话怎么说

如果你不想用任务卡，也可以直接在底部的输入框里用大白话说需求，就像给同事交代活儿一样。例如：

- 「把这个文件夹里的所有 Excel 合并成一张总表，存成 `汇总.xlsx`。」
- 「读一下这个项目，用几句话告诉我它是干嘛的。」
- 「帮我看看这几张扫描件上写了什么。」

按 **Enter** 发送，**Shift+Enter** 换行。发送后 AI 会一边说它在做什么、一边调用本机工具干活，右侧还能看到它动了哪些文件。

## 2. 核心概念（人话）

只需要理解这几个词，你就懂了如意工作台的全部心智。

### 工作文件夹 —— AI 只看这里

「工作文件夹」是你指定给 AI 的**唯一活动范围**。AI 读文件、写文件、找东西，全都在这个文件夹里进行；文件夹外面的东西它看不见也动不了。这是最重要的安全边界：**选对工作文件夹，就等于给 AI 划好了活动圈**。

顶栏左侧那个「📁 工作台」按钮显示当前工作文件夹的名字，鼠标停上去能看到完整路径。随时点它可以换一个文件夹。

### 会话 —— 一次完整的对话

「会话」就是你和 AI 的一段对话记录，像微信里的一个聊天窗口。左侧栏列出你所有会话，可以搜索、改名、删除、置顶。点「＋ 新会话」开一段新的。不同的活儿建议开不同会话，历史清楚、互不干扰。

### 安全四档 —— 你决定 AI 有多自由

顶栏中间有一枚 **🛡 安全** 按钮（叫「安全 chip」），显示当前的安全等级。点它弹出四张卡片，任选一档：

| 档位 | 什么意思 | 什么时候用 |
|---|---|---|
| **每步都问** | 每个操作先征得你同意，最稳妥 | 刚上手、或处理重要文件时。默认档。 |
| **小改动自动做** | 文件编辑自动执行，命令等敏感操作仍会问你 | 已经信任 AI 的改文件能力，但不想放开跑命令。 |
| **先计划再动手** | AI 先给你一份完整计划，你批准后它才执行 | 复杂任务，想先看清楚它打算怎么干再放行。 |
| **全自动** | 不再询问（警示样式） | 你完全清楚在做什么、且任务安全时。谨慎使用。 |

不确定就用「每步都问」，它最稳。切换随时生效，不影响正在进行的对话。

### 「都能撤销」—— 信任三件套

用如意工作台不用怕手抖，因为它有三层安全网，合起来就是一句话：**AI 改的东西，你都能看到、都能回退**。

- **检查点**：AI 每次改文件（写入 / 编辑 / 删除）之前，工作台会**自动**把文件的原样存一份。
- **审计**：右侧「审计」页签是一条时间线，AI 干过的每件事都记录在案，随时回看。
- **回滚**：想反悔时，可以把某一步、某一整轮的文件改动**撤销**，把文件恢复成改动前的样子。

所以安全弹层底部那句「文件改动都有检查点，可在审计中回看、可撤销」不是客套——它是真的。

## 3. 常见任务走查

下面每个任务都对应界面里的一张**任务卡**（首页那排卡片）。你既可以点任务卡填空，也可以直接照着「你可以这样说」的话术在输入框里说。

### 合并多个 Excel 表格 📊

把一个文件夹里的多个 xlsx 按列对齐合并成一张总表，还会自动加一列「来源文件」方便回溯、去掉完全重复的行。
> **你可以这样说**：「请把文件夹 `D:\报表\10月` 下的所有 Excel 表格合并成一个总表，保存为 `汇总.xlsx`。」

### 批量重命名文件 🏷️

按统一规则给一个文件夹里的文件改名（加前缀、编号、日期等）。AI 会**先给你一份「旧名→新名」对照表**让你过目、自检有没有重名冲突，确认后才真正动手——因为改名不能自动撤销，它会格外谨慎。
> **你可以这样说**：「把 `D:\照片\2026春游` 里的文件批量重命名，规则是：统一改成『春游-01、春游-02……』这样按修改时间排序编号。」

### PDF 文本汇总 📄

把一个文件夹里的多份 PDF 提取文本、逐份摘要，再汇总成一份总览（Markdown）。碰到扫描件（提不出文字的）它会单独标出来提示你去做 OCR。
> **你可以这样说**：「把 `D:\合同` 下所有 PDF 提取文本并汇总，结果写入 `合同摘要.md`。」

### 扫描件 OCR 转文字 🔍

把扫描件图片或扫描版 PDF 通过 OCR 识别成可编辑文字。**这个任务需要先启用「桌面控制」能力**（见第 5 章）；没启用时 AI 会明确告诉你，不会假装成功。
> **你可以这样说**：「对 `D:\扫描件` 里的图片和扫描版 PDF 做 OCR 识别，结果写入 `识别结果.md`。」

### 写周报草稿 📝

把你这一周的零散记录整理成条理清晰的周报（本周完成 / 进行中 / 下周计划 / 遇到的问题）。它**只归纳你提到的事，绝不编造**你没说的数据或成果。
> **你可以这样说**：「根据我这周的记录整理一份周报，写入 `周报.md`。这周做的事：跑通了新接口、修了两个登录 bug、周三开了需求评审、下周要出测试报告。」

### 导出 PDF（中文没问题）

让 AI 把整理好的内容导出成一份 PDF（周报、汇总、归档摘要都行）。**中文显示没问题**——导出时会自动挑选中文字体（优先微软雅黑，退而用宋体，再不行用内置的中文字体兜底），你不用操心乱码。
> **你可以这样说**：「把刚才这份周报导出成 PDF，标题写『第 27 周工作周报』。」

### 让 AI 帮我管版本（看改动 / 存版本）

如果你的工作文件夹是一个用 git 管理的项目，可以让 AI 帮你看有哪些改动、把当前进度存成一个版本。你不用记 git 命令，说人话就行。
> **你可以这样说**：「看一下现在改了哪些文件，跟我讲讲。」／「把当前改动存成一个版本，说明写『完成周报模板』。」

### 联网查资料（需先配置搜索服务）

如意工作台**默认全离线**，不主动联网。如果管理员在设置里配好了「联网搜索」服务，AI 才能上网查时效性资料（并且断网时会自动读本地缓存）。没配的话，这类请求它会告诉你「联网搜索当前不可用」。
> **你可以这样说**（已配好搜索服务时）：「查一下这个型号的最新报价，给我三个来源。」

## 4. 界面导览

界面分三块：左侧栏（会话）、中间（对话）、右侧（工具面板）。

### 顶栏（从左到右）

- **📁 工作文件夹**：当前工作区名字，点它换文件夹。
- **标题 / meta**：当前会话名和一些附加信息。
- **上下文电量**：一个小电池图标，表示当前对话「装了多少东西」。简易模式下只有快满了（≥60%）才显示。点它能看用量、也能手动「立即压缩」腾空间。
- **🛡 安全**：当前安全等级，点它切四档（见第 2 章）。
- **引擎**：当前用的是哪个 AI 引擎（Claude CLI 或某个 Provider）和哪个模型，点它切换。引擎旁边的小圆点还顺带显示联网状态。
- **⋯ 更多菜单**：主题（深 / 浅色）、界面模式（精简 / 专家）、能力矩阵、快捷键都收在这里。
- **🧰**：展开 / 收起右侧工具面板。

### 右侧常驻页签

- **文件**：当前工作文件夹的文件树。点一个文本文件能预览；也有「搜索项目」「读取文件」的小工具。
- **产物**：本次会话里 AI **生成的文件**（按轮次分组），可以预览、打开。做完一单活儿来这里拿结果最方便。
- **审计**：一条时间线，记录 AI 干过的每件事。点某一行能展开完整记录。可按来源、按类型过滤。
- **变更**：可回撤的文件改动（按轮次）。
- **Agent 工作流**：多 Agent 编排的运行监控画布。
- **用量**：本轮 token 与成本。

> 专家模式下，右侧还会多出一组「开发者」页签：终端、桌面、MCP、调试、存储。日常用不到，精简模式里默认藏起来。

### 输入区

底部就是你说话的地方。精简模式下很干净：**＋**（更多操作）、**📎**（添加文件）、**发送 ▷**。想调技能 / 命令面板可以直接在输入框按 `/`（专家模式里还会显示一个「⌘ 技能」按钮）。拖文件进来会变成附件，拖文件夹进来会问你要不要设为工作目录。

### 任务卡首页

没有正在进行的对话时，中间会显示任务卡（首页那排卡片）。精简模式下，任务卡是主要入口——点一张、填空、发送，最省心。

### 精简 ↔ 专家模式切换

在顶栏 **⋯ 菜单 →「界面」** 里切换（设置的「基础」页签里也能改）：

- **精简（人人可用）**：隐藏调试 / 终端 / MCP 等高级页签和按钮，工具卡显示「写入文件」这样的人话动词而非工具名，字大一点。**新装机默认精简。**
- **专家（完整界面）**：显示全部开发者页签和原始术语，适合懂技术的用户。

## 5. 设置指南（用户视角）

点左下角 **⚙ 设置** 打开设置窗口。这里说和你日常相关的几项。

### 引擎怎么配

如意工作台支持两类 AI 引擎，配好任意一个就能用：

- **Claude CLI**（设置 →「Claude CLI」页签）：如果你的电脑 / 内网已经装了 Claude 命令行工具，在「Claude CLI 路径」里填它的路径即可（通常管理员已经配好）。
- **API 服务商 Provider**（设置 →「Providers」页签）：如 DeepSeek、通义千问、智谱 GLM，或内网自建的模型服务。点「+ 添加 Provider」，选一个预设或手填地址，填上密钥即可。配好后在顶栏「引擎」里切到它。

> 密钥只保存在你本机，界面上显示时会打码（只露后 4 位），不会泄露。

### 联网搜索怎么开

设置 →「联网搜索」页签。选一个搜索服务（SearXNG 自建实例 / Bing / Brave / Tavily / 博查 Bocha / 自定义），填 Base URL 和 / 或 API 密钥。配好后 AI 就能联网查资料，断网时自动读本地缓存。**不配的话，工作台完全离线运行**，不影响本机文件类任务。

### 界面 / 输出风格

设置 →「基础」页签：

- **界面模式**：精简 / 专家（见第 4 章）。
- **输出风格**：详细（默认，会解释过程）／简洁（直接给结果，不啰嗦）。想让 AI 少说废话就选简洁。

## 6. 用得更顺手（技能库 / 工作台记忆 / 用量看板 / 工作流插话）

这几样不是必须，但用熟了会省很多事。都在界面上点点就能用，不用记命令。

### 技能库 —— 给 AI 装上「专长」

「技能」是一包写好的做事方法（比如「按公司规范审代码」「做 Office 自动化」）。启用某个技能后，AI 这一段对话里就会**自动带上这门专长**，需要时它会自己展开细则去照做，你不用每次都交代一遍。

- 打开：输入框按 `/`，或专家模式下点输入区的 **「⌘ 技能」** 按钮。
- 面板里分三组：**技能**、**命令**、**一键任务**（就是首页那些任务卡）。每张技能卡可打开“查看细则”，先看清适用场景、步骤、验收与停止条件。
- 技能有两个开关：**本会话启用**只影响当前对话；**设为常驻**会在所有对话里保持可用，适合你长期使用的文档、表格或开发规范。常驻技能仍采用按需展开，只把短索引放进提示词；不常用的专业技能建议保留为会话启用。
- 命令在 Claude CLI 引擎下插入 `/name`；在 Provider 引擎下会插入同一命令的完整任务模板，因此两种引擎都可使用。点击“查看模板”可先读完整步骤。
- 输入框旁的“技能 · N”会合并统计当前会话与常驻技能；不想要了可分别关闭。
- 一键任务的填写窗口可展开“查看完整执行说明”，确认步骤与验收标准后再开始。

### 浏览器与工具可以在对话里调整

默认情况下，AI 打开网址会使用你电脑的默认浏览器和已有登录状态，并在新标签页/窗口打开，不再自动弹出隔离的 Chrome for Testing。当前的如意工作台标签页受到保护：浏览器工具不会把它导航到外部网址、复用它或关闭它。需要网页元素级自动化时，可在 **设置 → 集成/MCP → 浏览器目标** 改为“连接已有浏览器（CDP）”或其他模式；显式选择“隔离测试浏览器”才会使用 Chrome for Testing。

如果硬件加速网页让 UIA 只能读到浏览器外壳，AI 会改用网页连接、OCR 或截图定位。它能读到画面和文字，但对纯 Direct3D 自绘界面不一定能恢复真正的按钮语义。

你也可以直接说“把浏览器改回默认浏览器”“添加这个 MCP”“停用某个连接器”。AI 会先列出现状和准备修改的内容；真正保存前仍会出现执行级权限确认，密钥值不会显示在对话里。

### 工作台记忆 —— 让 AI 记住你的习惯

如果有些偏好或经验你不想每次重说（「我们项目的命名规范是……」「导出报告一律用这个模板」），可以把它**存成一条工作台记忆**。之后在这个项目里开新对话，AI 会自动带上这些记忆当参考。

- **存**：一段对话里得到有用的结论后，点助手消息上的 **「存为记忆」**（AI 会先帮你拟一稿，你改完确认再入库——不会偷偷记）。也可以在记忆面板里手写一条。
- **用**：同一个工作文件夹（项目）下的记忆会**自动启用**；全局记忆则由你手动开。
- **管**：记忆面板里能看列表、编辑、删除、启停，还能把别的项目的记忆「迁移到当前项目」。
- 它和项目里的 `CLAUDE.md` 分工清楚：**仓库约定**（团队共享的规矩）写进 `CLAUDE.md`，**你个人的习惯/经验**放工作台记忆。

### 用量看板 —— 花了多少、用了多少 token，一目了然

顶栏或菜单进 **「用量」** 页，能看到 token 消耗和花费，按引擎、按 Provider、按会话、按天分组。

- **诚实计费**：如果你的 Claude CLI 接的是**第三方 Coding Plan**（比如火山方舟 Ark 这类按月套餐、不按 token 计费的端点），看板不会瞎算美元金额，而是标注「计划内计费」——只记 token 数，不假装知道你花了多少钱。只有你自己配了单价、或用官方按量计费时，才显示可信的花费。
- 工作流里子代理的消耗、自动压缩对话的消耗，**也都算进来**，不漏账。
- 可以设**月度预算**，快超了会提醒你。

### 工作流里给某个 Agent 插话 / 批准新任务

当你让 AI 跑一个**多步工作流**（多个 Agent 协作）时，右侧能实时看到每个节点在干嘛。两个实用操作：

- **定向插话**：某个节点正在跑、你想补一句（「顺便也检查一下并发安全」），点那个节点的 **「插话」**，你的话会在它下一步之前送到——不用打断整个流程。
- **批准新任务**：干活的 Agent 中途可能发现「还需要加一步」，于是提议一个新任务。这时会冒出一张**待批准卡**，写清**谁提议、要做什么、大概花多少**，你点「同意添加」它就加进流程，点「不用了」就算了。主动权始终在你手上。

## 7. 常见问题

**Q：顶栏引擎显示没就绪 / 首跑卡说「还没有可用的 AI 引擎」，怎么办？**
去设置里配一个引擎（第 5 章）：要么填 Claude CLI 路径，要么加一个 Provider 填上密钥。配好后引擎旁的小圆点会变绿。拿不准就找管理员。

**Q：AI 弹出一个「权限」窗口问我允不允许，那是什么？**
这是安全机制。在「每步都问」「小改动自动做」等档位下，AI 要做敏感操作（改文件、跑命令）前会先弹窗征求你同意。看清楚它要做什么，允许或拒绝即可；也可以选「本工具以后自动允许」。窗口会显示这步的风险等级和能不能撤销。如果你一直不理它，超时后会**自动拒绝**（不会替你放行）。

**Q：AI 改错了 / 改了我不想改的文件，怎么撤销？**
右侧「审计」页签能看到它改了什么。工作台在每次改文件前都自动存了检查点，所以你可以把某一步或某一整轮的文件改动回滚，恢复成改动前的样子。这就是「信任三件套」（第 2 章）。注意：**批量重命名这类操作 AI 会先给对照表让你确认**，因为改名本身不属于可自动撤销的范畴。

**Q：断网了还能用吗？**
能。如意工作台**默认就是全离线的**：本机的文件操作、脚本、PowerShell、桌面控制、读写 Office、导出 PDF、OCR……这些都在你自己电脑上跑，不需要网。只有「联网搜索」这一类需要网络；而且就算断网，之前搜过的资料还能从本地缓存里读到。

**Q：导出的 PDF 里中文会乱码吗？**
不会。导出时会自动按「微软雅黑 → 宋体 → 内置中文字体 → 拉丁兜底」的顺序挑字体，中文正常显示。极个别没装中文字体的机器上，会退到内置的中文字体，阅读器侧照样能正确显示中文。

**Q：对话越来越长，感觉变慢 / 顶栏电量快满了？**
点顶栏的「上下文电量」，里面有「立即压缩」——它会让 AI 把前面的对话概括压缩一下，腾出空间，之后继续聊不受影响。快满时工作台有时也会自动帮你压缩。
